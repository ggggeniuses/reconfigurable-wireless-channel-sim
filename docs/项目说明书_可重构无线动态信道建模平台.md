# RIS/FAS 辅助动态信道建模与海上通信扩展仿真平台

## 项目定位

本项目是一个 MATLAB 研究型仿真平台，用于分析 RIS/FAS 可重构无线系统中的
动态信道、传播统计、波束域稀疏性、建模复杂度和信道容量。

项目包含三个主要场景：

1. RIS-V2V：RIS 辅助车联网信道的空间、时间和频率相关特性。
2. FAS-UAV：流体天线辅助无人机对地通信中的动态端口选择和容量分析。
3. Maritime UAV-RIS-Shipborne-FAS：融合空中 RIS、海面散射和船载 FAS
   端口选择的海上无人机通信扩展场景。

项目重点是形成结构清晰、可运行、可测试并能重复生成结果的科研工程。海上
扩展模块用于定性趋势分析，目前不宣称已经过实测标定。

## 已实现功能

### RIS-V2V

- Spatial CCF、Temporal ACF 和 Frequency FCF。
- RIS 阵列尺寸和单元间距扫描。
- 不同车辆运动状态下的相关性分析。
- 阵列域与波束域的理想容量不变性检查。
- BDCM 能量集中度、稀疏性和计算复杂度分析。

### FAS-UAV

- 动态 LoS/NLoS 信道生成。
- 流体天线有效端口选择。
- 端口间距和有效端口数量对应的建模误差。
- 不同天线孔径和 UAV 运动状态下的容量分析。
- FAS 与传统 ULA 的容量比较。

### Maritime UAV-RIS-Shipborne-FAS

- UAV、空中 RIS、船舶和海面散射体的三维几何建模。
- RIS 级联相干路径和海面散射 NLoS 路径。
- 链路距离、Rician 因子和 RIS 位置对 Temporal ACF 的影响。
- 相关船载 FAS 端口的选择增益和容量分析。

## 运行方式

完整运行：

```matlab
run_all
run_tests
```

分别运行：

```matlab
run_benchmarks
run_extensions
run_fas_uav
run_fas_ship
```

高采样配置：

```powershell
$env:FAS_UAV_PROFILE = "final"
$env:FAS_SHIP_PROFILE = "final"
matlab -batch "run_all"
matlab -batch "run_tests"
```

## 验证范围

- DFT 波束形成矩阵酉性与容量一致性。
- 固定随机种子下的信道确定性。
- 有效端口索引和信道矩阵尺寸。
- 建模误差、稀疏性和复杂度指标。
- 海上 ACF 的归一化、有限值与取值范围。
- 可选端口增加时 FAS 选择容量的单调性。
- 输出文件完整性和 MAT 配置路径可移植性。
