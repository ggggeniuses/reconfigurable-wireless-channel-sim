# Resume Description

## Chinese Version

**RIS 辅助 V2V 三维动态信道建模与波束域相关性仿真平台**

基于 MATLAB 构建 RIS 辅助 V2V 三维动态信道建模项目，整理并重构原始 STF-CF 代码，
复现 GBSM/BDCM 下的空间互相关、时间自相关和频率相关特性；进一步扩展分析 RIS
阵列尺寸、单元间距和车辆运动状态对信道非平稳性的影响，形成可复现实验脚本、结果图
和 GitHub 项目文档。

## English Version

**RIS-V2V Beam-Domain Channel Modeling Simulator**

Developed a MATLAB simulation platform for RIS-aided V2V 3D dynamic channel
modeling. Refactored original STF-CF code to reproduce GBSM/BDCM spatial CCF,
temporal ACF, and frequency FCF characteristics, and further analyzed the
impact of RIS array size, element spacing, and mobility states on channel
non-stationarity.

## Interview Talking Points

- Explain why RIS-aided V2V channels are non-stationary.
- Explain the difference between GBSM and BDCM.
- Explain what CCF, ACF, and FCF measure.
- Explain why BDCM is useful for large MIMO/RIS channel modeling.
- Distinguish strict reproduction outputs from extension experiments.
